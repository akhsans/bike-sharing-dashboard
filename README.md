# 📊 Bike Sharing Dashboard

Dashboard ini dibuat untuk menganalisis data **Bike Sharing** menggunakan Python dan Streamlit. Dashboard ini menampilkan berbagai visualisasi interaktif untuk memahami pola penggunaan sepeda.

---

## 🚀 Cara Menjalankan Dashboard

Ikuti langkah-langkah berikut untuk menjalankan dashboard di komputer kamu:

### 1. Clone Repository

```bash
git clone https://github.com/username/nama-repo.git
cd nama-repo
```

### 2. Install Library yang Dibutuhkan

Pastikan kamu sudah install Python, lalu jalankan:

```bash
pip install -r requirements.txt
```

Jika belum ada file `requirements.txt`, bisa install manual:

```bash
pip install streamlit pandas matplotlib
```

---

### 3. Jalankan Dashboard

```bash
streamlit run dashboard.py
```

---

### 4. Buka di Browser

Setelah dijalankan, dashboard akan otomatis terbuka di browser:

```
http://localhost:8501
```
